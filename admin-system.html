Panduan Seru untuk Jadi Admin Sistem dan Menguasai Linux

Hei, calon admin! Siap untuk menjelajahi dunia sistem administrasi dan Linux dengan cara yang lebih fun? Yuk, kita mulai petualangan ini!

Dasar Ilmu System Administrator

Bayangkan kamu adalah kapten dari kapal luar angkasa yang disebut 'Server'. Tugasmu? Pastikan kapal ini terbang mulus tanpa drama:

  • Manajemen User: Kamu adalah penjaga gerbang. Siapa boleh masuk, siapa yang cuma bisa ngintip dari luar.
  • Backup dan Recovery: Ini seperti save game, tapi kalau gagal, ya, bisa kiamat kecil.
  • Keamanan: Pakai jubah invisibility untuk servermu dari para hacker yang nakal.
  • Monitoring: Seperti jadi dokter untuk server, selalu cek denyut nadinya.

Dasar Linux

Linux itu seperti Lego untuk orang dewasa. Kamu bisa bangun apa saja, tapi pertama, kenali dulu potongan-potongannya:

  • Distro: Ini seperti rasa es krim. Ada Ubuntu, Fedora, Debian, dan lainnya. Pilih yang sesuai selera!
  • Shell: Tempat kamu bisa jadi penyihir dengan perintah-perintah ajaib.
  • File System: Pikirkan seperti perpustakaan besar, tapi kamu harus tahu di mana buku-bukunya disimpan.

Command Dasar Linux untuk Admin Sistem

Ini dia beberapa mantra yang wajib kamu hafal:

  • ls - List, bukan lari, tapi lihat isi direktori. Seperti buka lemari, lihat bajunya.
  • cd - Change Directory. Jalan-jalan di sistem filemu.
  • mkdir - Make Directory. Bikin folder baru, buat rumah baru untuk file-filemu.
  • rm - Remove. Hati-hati, ini seperti mantra penghapus, bisa bikin file hilang selamanya!
  • chmod - Change Mode. Atur siapa yang boleh baca, tulis, atau eksekusi file. Ini seperti atur izin masuk ke klub.
  • ps - Process Status. Cek siapa saja yang lagi kerja di sistemmu.
  • top - Lihat proses yang lagi makan banyak resource. Siapa yang rakus, kita tahu!
  • sudo - Superuser Do. Kamu jadi superhero di sini. Segala bisa, tapi hati-hati, power besar tanggung jawab besar.
